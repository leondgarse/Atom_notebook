### **Phase 2: Data Ingestion (Python)**
Next, we load the segmented K-Means dataset and push it into the graph database, translating tabular rows into connected nodes.

**1. The Ingestion Script**
Execute this block in Jupyter. It connects to Neo4j, reads your CSV, and uses a parameterized Cypher query to build the `Customer`, `Tribe`, and `Gender` nodes, along with their relationships.

```python
import pandas as pd
import io
from neo4j import GraphDatabase
from sklearn.preprocessing import StandardScaler

# ==========================================
# 1. GENERATE AND PREPROCESS THE DATA
# ==========================================
df_final = pd.read_csv('Mall_Customers.csv')

# Rename 'Genre' to 'Gender'
df_final.rename(columns={'Genre': 'Gender'}, inplace=True)

# Standardize the features
scaler = StandardScaler()
scaled_cols = scaler.fit_transform(df_final[['Age', 'Annual Income (k$)', 'Spending Score (1-100)']])

# Add the scaled features back to the DataFrame
df_final['Age_Scaled'] = scaled_cols[:, 0]
df_final['Income_Scaled'] = scaled_cols[:, 1]
df_final['Score_Scaled'] = scaled_cols[:, 2]

# Add a mock Cluster Name for the Tribe nodes
# (In your full project, this comes from your K-Means results)
def assign_mock_cluster(score):
    return "VIPs" if score > 50 else "Conservatives"

df_final['Cluster_Name'] = df_final['Spending Score (1-100)'].apply(assign_mock_cluster)

print("--- Preprocessed DataFrame ---")
display(df_final)
print("\n")

# ==========================================
# 2. NEO4J BATCH INGESTION
# ==========================================
# Connect to the local Docker database
URI = "bolt://localhost:7687"
AUTH = ("neo4j", "password")
driver = GraphDatabase.driver(URI, auth=AUTH)

def load_mall_data(tx, row):
    cypher_query = """
    MERGE (t:Tribe {name: $tribe_name})
    MERGE (g:Gender {name: $gender})

    MERGE (c:Customer {id: $customer_id})
    SET c.age = $age,
        c.income = $income,
        c.score = $score,
        c.age_scaled = $age_scaled,
        c.income_scaled = $income_scaled,
        c.score_scaled = $score_scaled

    MERGE (c)-[:BELONGS_TO]->(t)
    MERGE (c)-[:IDENTIFIES_AS]->(g)
    """

    tx.run(cypher_query,
           customer_id=row['CustomerID'],
           age=row['Age'],
           income=row['Annual Income (k$)'],
           score=row['Spending Score (1-100)'],
           age_scaled=row['Age_Scaled'],       # Uncommented!
           income_scaled=row['Income_Scaled'],
           score_scaled=row['Score_Scaled'],
           tribe_name=row['Cluster_Name'],
           gender=row['Gender'])

# Execute the batch ingestion
with driver.session() as session:
    for index, row in df_final.iterrows():
        session.execute_write(load_mall_data, row)

print("Graph ingestion complete. You can now view the nodes in http://localhost:7474")
driver.close()
```

### **Phase 3: Graph-Native Machine Learning (Neo4j Browser)**
With the data loaded, switch to the Neo4j Browser (`http://localhost:7474`). We execute algorithms natively inside the database to find mathematical communities.

**1. Create the K-Nearest Neighbors (KNN) Network**
Project the scaled features into memory and calculate Euclidean distances to link every customer to their 5 closest behavioral peers.
```cypher
// A. Project nodes into memory
CALL gds.graph.project(
  'mall_graph_knn',
  'Customer',
  '*',
  /* { nodeProperties: ['age_scaled', 'income_scaled', 'score_scaled'] } */
  { nodeProperties: ['income_scaled', 'score_scaled'] }
);

// B. Execute KNN and write 'IS_SIMILAR_TO' relationships
CALL gds.knn.write('mall_graph_knn', {
    /* nodeProperties: ['age_scaled', 'income_scaled', 'score_scaled'], */
    nodeProperties: ['income_scaled', 'score_scaled'],
    writeRelationshipType: 'IS_SIMILAR_TO',
    writeProperty: 'similarity_score',
    topK: 5
});
```

**2. Execute Louvain Community Detection**
Now, find the dense "pockets" within that newly created web of similarities.
```cypher
// A. Drop old graph and project the new one with relationships included
CALL gds.graph.drop('mall_graph_knn', false);

CALL gds.graph.project(
  'mall_graph_louvain',
  'Customer',
  { IS_SIMILAR_TO: { orientation: 'UNDIRECTED', properties: 'similarity_score' } }
);

// B. Execute Louvain and stamp nodes with their new Community ID
CALL gds.louvain.write('mall_graph_louvain', {
    relationshipWeightProperty: 'similarity_score',
    writeProperty: 'louvain_community'
});
```
