    - **特殊单词拼写** 如 `ウォ` - `who`，`ティ` - `thi`，`ディ` - `dhi`

    - **IME Pad** 手写输入，用于输入陌生汉字，使用时右键单击日语输入法标志
## 声调类型
  - 单词的第一个音节和第二个音节的音高一定不同
  - 一个单词的声调，或没有下降，或只有一处下降
  - 尾高词后面的助词要降调
## 音读训读
  - 汉字读音分为 **音读 - おんどく(音読)** / **训读 - くんどく(訓読)**，有的汉字有 10 个以上读音
  - **音读** 一般是汉语传入的，有多音字，**训读** 是日本已有的汉字
  - **音读** 发音会有一些规律
    - **n -> ん** / **ng -> い**，先生 xian sheng -> さん　さい，店员 dian yuan -> たん　いん，声 sheng -> せい
    - **iu -> ゆ**，留 liu -> りゅう，牛 niu　->　ぎゅう，求 qiu -> きゅう
## 送假名
  - **送假名 おくりがな(送り仮名)** 用于辨别汉字读音，跟在汉字后面的假名
  - 如汉字 `祭` 有两个读音 `さい` / `まつり`，使用 `祭り` 表示读作 `まつり`

## 动词分类

## 动词 ます 形

    | い  | き  | し  | ち  | に  | ひ  | み  | い  | り  | ぎ  | じ  | び  | ぴ  |

## 简体形
  - **敬体形** 使用 **～ます** / **～です** 的形式，一般是对长辈或不太亲近的人使用
  - **简体形** 对同辈或晚辈，或者关系较亲密时则使用简体形
  - **动词的简体形** 使用动词原形 / ない 形 / た 形 / なかった 形
    | **动词**                | 敬体形                             | 简体形                                      |
    | ----------------------- | ---------------------------------- | ------------------------------------------- |
    | 现在时 / 将来时肯定形式 | かいます(買います)                 | かう(買う) 动词原形                         |
    | 现在时 / 将来时否定形式 | かいません(かいません)             | かわない(買わない) 动词 ない 形             |
    | 过去时肯定形式          | かいました(買いました)             | かった(買った) 动词 た 形                   |
    | 过去时否定定形式        | かいませんでした(買いませんでした) | かわなかった(買わなかった) 动词 なかった 形 |

    - 疑问句的简体形，使用 **动词原型升调**
    - 简体形式通常可以省略助词，如 を / か / は
    - **ある(有る)** 现在时 / 将来时否定形式的简体形是 **ない**
  - **一类形容词谓语形式的简体形** 去掉敬体形的 **です**
    | **一类形容词谓语形式**  | 敬体形                             | 简体形                       |
    | ----------------------- | ---------------------------------- | ---------------------------- |
    | 现在时 / 将来时肯定形式 | いそがしいです(忙しいです)         | いそがしい(忙しい)           |
    | 现在时 / 将来时否定形式 | いそがしく(忙しく)　ないです       | いそがしく(忙しく)　ない     |
    | 过去时肯定形式          | いそがしかったです(忙しかったです) | いそがしかった(忙しかった)   |
    | 过去时否定定形式        | いそがしく(忙しく)　なかったです   | いそがしく(忙しく)　なかった |

  - **二类形容词谓语形式的简体形**
    | **二类形容词谓语形式**  | 敬体形                             | 简体形                       |
    | ----------------------- | ---------------------------------- | ---------------------------- |
    | 现在时 / 将来时肯定形式 | かんたんです(簡単です)             | かんたんだ(簡単だ)           |
